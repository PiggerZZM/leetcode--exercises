# leetcode-exercises

组队开刷LeetCode，欢迎加入

## 提交规范（必读）

1. 一道题一个文件夹，题目文件夹命名为`LeetCode[题号]`
2. 可以提交任何一种语言的代码
3. 请务必写题解，题解存放在题目文件夹里的`README.md`
4. 代码应以合适的方式注明"解题人"，"提交时间"等必要的信息，题解也需如此
5. 如果使用IDE，请注意修改`.gitignore`。请勿提交配置文件、编译后的二进制文件等。

## 题解规范（必读）

题解应存放在对应题目文件夹里的`README.md`

题解内容需要包含：

1. 题目简要概述
2. 题解的小标题并注明相关的算法标签
3. 题解的正文
4. 可选的插图，表格等其他辅助说明
5. 小标题旁注明提交人，提交时间
